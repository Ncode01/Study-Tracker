# Study Tracker - O/L Exam Preparation App

A comprehensive study planning application designed for Sri Lankan G.C.E. O/L students preparing for exams in February 2026.

## Features

### Dashboard
- Countdown to exam dates
- Quick overview of today's tasks
- Subject progress at a glance
- Quick-add session button

### Study Calendar
- Interactive calendar with month/week/day views
- Drag-and-drop session management
- Color-coded subjects
- Quick session scheduling from time slots

### Task Management
- Create tasks with due dates and priorities
- Recurring tasks support (daily, weekly, custom)
- Subject-categorized tasks
- One-click completion toggle

### Pomodoro Timer
- Customizable work/break intervals
- Multiple presets (Focus mode, Short sessions, Extended study)
- Subject tracking for sessions
- Session count and progress tracking

### Progress Tracking
- Mark entry for each subject
- Progress charts showing improvement over time
- Subject-wise statistics
- Target tracking

## Tech Stack

- React 19 - UI framework
- TypeScript 5.9 - Type safety
- Vite (rolldown-vite) - Build tool
- date-fns - Date manipulation
- react-big-calendar - Calendar component
- Recharts - Data visualization
- CSS Modules - Scoped styling

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

npm install
npm run dev
npm run build
