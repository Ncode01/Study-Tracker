# Styles Directory

## Purpose
Contains global styles, CSS variables, and shared style utilities.

## Contents
- `global.css` - Global CSS reset and base styles
- `variables.css` - CSS custom properties (colors, spacing, etc.)
- `animations.css` - Shared animation keyframes

## Usage
Global styles are imported in main.tsx. CSS Modules are used for component-specific styles.

## Design System
- Primary Color: #4F46E5 (Indigo)
- Success Color: #10B981 (Emerald)
- Warning Color: #F59E0B (Amber)
- Error Color: #EF4444 (Red)
- Spacing: 4px base unit
- Border Radius: 8px default

## Guidelines
- Use CSS variables for consistency
- Mobile-first responsive design
- Desktop-optimized layouts
